# COVID-Smart-Elevator

Design of a COVID Smart Elevator, which uses hand gesture recognition to replace buttons inside the elevator and body identification to monitor the number of people inside the elevator at once. 
Programmed with OpenCV, Python
#
Hand Gesture Recognition:
1. Background Subtraction
2. Contour Hull
3. Convexity Defects
#
Body Identification:
Uses OpenCV's pretrained Haar Cascade Classifier
